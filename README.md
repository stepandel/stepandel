# Stepan Arsentjev

Engineer, product builder, systems architect.

I enjoy building great products that solve real pain points for others or myself. Recently, I've been building agentic harnesses for real-world applications using the latest AI tech, with a focus on tools that can actually do work: answer calls, operate on incidents, ship code, route tasks, sell, buy, book, and coordinate.

I have also spent the last few years building production crypto governance infrastructure, data systems, and developer tools. I like small teams, simple architectures, and software that survives contact with the messy real world.

## Agent Harnesses

- [Assemblio](https://github.com/stepandel/assemblio) - a self-hostable platform for running fleets of self-modifying AI agents with isolated per-agent data planes scheduled by Nomad.
- [Discovery Calls Co-Pilot](https://github.com/stepandel/sales-co-pilot) - a local desktop co-pilot that listens to discovery calls and surfaces better questions in real time.

- [AgentMail CLI](https://github.com/stepandel/agentmail-cli) - a terminal tool for creating and managing AgentMail inboxes and messages in agent workflows.
- [Clawup](https://github.com/stepandel/clawup) - a secure deployment path for OpenClaw agent swarms using Docker sandboxes and a Tailscale mesh.
